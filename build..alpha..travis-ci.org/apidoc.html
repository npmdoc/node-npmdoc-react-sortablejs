Use <b>&lt;ReactSortable /&gt;</b> instead of <b>&lt;Sortable /&gt;</b>
; in your JSX code since the Sortable library will export a <b>window.Sortable</b> object if you're running
 JSX code directly in the browser. For example:
```js
<ReactSortable
    tag="ul"
    onChange={(order) =>
        this.props.onChange(order);
    }}
>
    {items}
</ReactSortable>
```
